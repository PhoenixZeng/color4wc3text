# color4wc3text

这是一个vscode拓展

使得魔兽争霸3的字符串前缀可以被颜色选择器识别和修改

支持以下语法

``` txt
    war3
         - |cffffffff
    color
         - color(255,255,255)
         - color(255,255,255,255)
    css
         - #ffffff
```

目前支持以下后缀格式的文件

    war3
        - jass
        - lua
        - ini
        - vjass
        - zinc
        - fdf
        - json

    color
        - lua
        - vjass
        - zinc

    css
        - html
        - css
        - xml
        - json
